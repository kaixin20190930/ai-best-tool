# Product Overview

**Tap4 AI Web UI** is an open-source AI tools directory platform that helps users discover, explore, and submit AI tools. The platform is designed for easy deployment and customization.

## Core Features

- **AI Tools Directory**: Browse and discover AI tools across multiple categories (productivity, design, chatbots, etc.)
- **Multi-language Support**: Full internationalization with 9 languages (en, cn, tw, de, es, fr, jp, pt, ru)
- **User Features**: Authentication, favorites, ratings, comments, and notifications
- **Admin Dashboard**: Tool management, analytics, and content moderation
- **SEO Optimized**: Comprehensive SEO with structured data, hreflang, and performance optimization
- **Tool Submission**: Public form for users to submit new AI tools

## Target Audience

- Users searching for AI tools
- AI tool creators wanting visibility
- Developers building similar directories

## Business Model

- Google AdSense integration
- Sponsored tool listings
- Premium placements on startup page
